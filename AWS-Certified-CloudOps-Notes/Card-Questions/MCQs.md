### AWS Runbooks
- Runbooks usually refer to the **AWS Systems Manager Automation documents**.
- These are prebuilt or custom workflows that performs operational tasks like patching, instance recovery, AMI creation, etc.

#### Common EC2 related runbooks
- AWS-StopEC2Instance
- AWS-StartEC2Instance
- AWS-RestartEC2Instance
- AWS-CreateImage
- AWS-UpdateLinuxAMI
- AWS-UpdateWindowsAMI
- AWS-RunPatchBaseline
